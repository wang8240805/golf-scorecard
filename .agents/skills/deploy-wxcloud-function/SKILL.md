# 部署微信云函数技能

**用途：** 修改OCR代码后，自动压缩上传部署到微信云开发云函数，并验证部署成功。

## 触发方式：
`/deploy-wxcloud`

## 执行步骤：

1. **验证代码语法** - 检查OCR代码是否有语法错误
2. **压缩云函数目录** - 将 `cloudfunctions/ocr-scorecard` 压缩为 zip 包
3. **检查云函数配置** - 验证 `config.json` 存在
4. **输出部署指南** - 给出微信开发者工具上传步骤
5. **验证最终结果** - 列出需要检查的关键点

## 输出格式：

```
## ☁️ 微信云函数部署准备完成

### 检查结果
| 检查项 | 状态 |
|--------|------|
| 代码语法检查 | ✅ 通过 |
| 云函数目录 | ✅ 存在 |
| index.js | ✅ 完整 |
| package.json | ✅ 存在 |
| config.json | ✅ 存在 |

### 上传步骤（微信开发者工具）
1. 打开 云开发 → 云函数 → 找到 `ocr-scorecard`
2. 右键 → 上传更新云端部署
3. 选择 "上传并部署：云端安装依赖"
4. 等待上传完成 → 部署成功

### 验证要点
- [ ] 识别后洞数 = 18
- [ ] 总标准杆 = 72
- [ ] 不会提取上方码数数字
- [ ] 不补全不推断，只返回真实识别结果
```
