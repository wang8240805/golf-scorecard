# JavaScript 代码审查技能

**用途：** 修改 JavaScript 代码后，自动运行代码审查，检查常见问题。

## 触发方式：
`/js-review [file-path]`

## 检查项：

### 语法错误
- [ ] 括号匹配正确
- [ ] 引号匹配正确
- [ ] 逗号使用正确
- [ ] 没有重复的函数声明

### 代码规范
- [ ] 缩进一致（项目使用 2 空格）
- [ ] 双引号字符串（符合项目规范）
- [ ] 函数使用 ES5 function 语法（符合项目规范）

### 安全检查
- [ ] 没有硬编码的 API Key 或密钥
- [ ] 所有异步操作有错误处理
- [ ] 外部输入有验证

### 性能检查
- [ ] 没有无限循环风险
- [ ] 避免不必要的重复计算

### 错误处理
- [ ] Promise/Sync 错误都有 catch
- [ ] 用户可见错误有友好提示

## 输出格式：

```
## JavaScript 代码审查报告

### 文件: {file}

| 检查项 | 状态 | 问题 |
|--------|------|------|
| 语法正确 | ✅ 通过 | - |
| 缩进一致 | ✅ 通过 | - |
| ... | ... | ... |

### 发现问题：

{如果有问题，列出问题位置和修复建议}

### 结论：ready to merge / 需要修复
```
